## Usage

+ Init

  ```shell
  #第一种方式：
  git clone https://github.com/didadidaboom/hexo-theme-wublog.git
  cd hexo-theme-wublog
  npm install
  
  #第二种方式：
  git clone https://github.com/didadidaboom/hexo-theme-wublog/theme/wublog theme/wublog
  ```
